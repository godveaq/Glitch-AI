# Glitch AI

Glitch AI is a terminal-based AI assistant with typing animations and capabilities for coding, chatting, and providing information. It features a distinctive glitch-style logo and typing animations that make interactions feel more dynamic and engaging.

## Features

- **Typing Animations**: All AI responses are displayed with realistic typing animations
- **Coding Assistance**: Provides code examples, explanations, and programming concepts
- **Information Provision**: Answers general questions and provides knowledge on various topics
- **Math Calculations**: Performs basic mathematical operations
- **Clean Output Formatting**: Beautifully formatted responses and code blocks
- **Glitch-style Logo**: Animated logo with occasional glitch effects

## Capabilities

- Programming language explanations (Python, JavaScript, Java, C++)
- Code examples for various programming languages
- Programming concepts and best practices
- Basic math calculations
- General knowledge and conversational abilities
- Error explanation and debugging tips

## Usage

Run the application using Python:

```bash
python glitch_ai.py
```

The AI will greet you with an animated logo and then wait for your input. You can ask about programming, request code examples, ask general questions, or perform calculations.

Type `exit`, `quit`, or `bye` to end the conversation.

## Examples

- "Show me a Python example"
- "What is recursion?"
- "Calculate 5 + 3"
- "What is object-oriented programming?"
- "Explain JavaScript functions"

## Technical Details

- Built with Python 3
- No external dependencies required
- All animations are implemented with native Python functions
- Clean, organized codebase with clear separation of concerns
